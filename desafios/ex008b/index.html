Outras formatações

Código-fonte / pré-formatações

Para inserir um trecho de código-fonte ou texto que não deve ser formatado, utilize a tag <pre>. O conteúdo dentro dessa tag preserva espaços e quebras de linha.

function exemplo() {
    console.log("Este é um exemplo de código-fonte.");}

Comentários

Comentários em HTML são inseridos utilizando a sintaxe <!-- comentário -->. Eles não são exibidos na página, mas podem ser úteis para documentar o código.

Exemplo de comentário:

<!-- Este é um comentário em HTML -->

Entidades HTML

Entidades HTML são usadas para representar caracteres especiais que têm significado em HTML, como <, > e &. Aqui estão algumas entidades comuns:

  • &lt; para o símbolo menor que (<)
  • &gt; para o símbolo maior que (>)
  • &amp; para o símbolo e comercial (&)
  • &quot; para aspas duplas (")
  • &apos; para aspas simples (')
gostoso de usar

Exemplo de uso de entidades:

<p>5 &lt; 10 e 10 &gt; 5</p>

Isso será exibido como: 5 < 10 e 10 > 5
