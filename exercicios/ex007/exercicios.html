Exercícios de HTML para Iniciantes

  1. Crie um parágrafo com seu nome e utilize a tag correta para destacar o nome.
  2. Adicione um título principal (usando <h1>) e um subtítulo (usando <h2>) à página.
  3. Insira uma lista não ordenada com três hobbies seus.
  4. Adicione uma imagem de sua escolha usando a tag <img>.
  5. Crie um link para um site que você gosta, usando a tag <a>.
  6. Utilize a tag <strong> para deixar uma palavra em negrito em um parágrafo.
  7. Adicione um rodapé simples usando a tag <footer> com seu e-mail de contato.
  8. Use a tag <section> para separar um conteúdo sobre “Sobre mim”.
  9. Insira um endereço diferente usando a tag <address>.
  10. Crie uma tabela simples com duas colunas: “Nome” e “Idade”.
